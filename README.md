# Dev Container Features

A repository for implementing and testing Dev Container Features.

## Structure

```
src/
  <feature-id>/
    devcontainer-feature.json   # Feature definition (options, mounts, etc.)
    install.sh                  # Install script executed at container build time
    README.md                   # Per-feature documentation
test/
  <feature-id>/
    scenarios.json              # Scenario test definitions
    default.sh                  # Tests for default configuration
    <scenario>.sh               # Tests for each scenario
    test.sh                     # Script for auto-generated tests
```

## Features

| Feature | Description |
|---------|-------------|
| [`persistence`](src/persistence/README.md) | Persists user-space data such as AI tool configurations and credentials using a Docker volume |

## Testing

Tests are run using [`@devcontainers/cli`](https://github.com/devcontainers/cli).

```bash
# Auto-generated test (build and verify persistence with default options)
devcontainer features test --skip-scenarios -f persistence -i mcr.microsoft.com/devcontainers/base:ubuntu .

# Scenario tests (all scenarios defined in scenarios.json)
devcontainer features test -f persistence --skip-autogenerated .

# Global scenarios (global tests under test/)
devcontainer features test --global-scenarios-only .

# Run all tests
devcontainer features test -f persistence .
```

### Prerequisites

- Docker must be running
- `@devcontainers/cli` must be installed (`npm install -g @devcontainers/cli`)
