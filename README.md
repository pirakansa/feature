# Dev Container Features

`features/` は Dev Container Features の実装とテストを管理するディレクトリです。

## 構成

- `src/`: 各 Feature の実装
- `test/`: 各 Feature のシナリオテスト

現在の Feature:

- `persistence`: ユーザー領域データを永続化する Feature
- `hello`: テスト/サンプル用 Feature

## テスト

`@devcontainers/cli` を使って実行します。

```bash
# 自動生成テスト（シナリオ除外）
devcontainer features test --skip-scenarios -f persistence -i ubuntu:latest .

# シナリオテスト
devcontainer features test -f persistence --skip-autogenerated --skip-duplicated .

# グローバルシナリオ
devcontainer features test --global-scenarios-only .
```
